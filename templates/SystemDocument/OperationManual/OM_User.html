使用者說明

  • 當以訪客身分進入時，僅能查看公告及系統相關介紹。
  • 當使用者登入後，在上方導覽列會顯示該使用者所擁有的權限（此圖為擁有所有權限之使用者）。
  • 權限區分為：
    • SystemManager：系統管理員
    • TestManager：考試管理員
    • TBManager：題庫管理員
    • TBOperator：題庫操作員
    • Viewer：成績檢閱者
    • Testee：受測者
  • 點擊右上角的「使用者」圖示，即可查看一般使用者功能表。
  • 使用者選單說明，如下：
    • User Profile：檢視個人資料
    • Profile Update：更新個人資料
    • Email：電子郵件
    • Password Change：更改密碼
    • Report：問題回報
    • About：關於系統
  • 點擊「Logout」可登出系統，點擊「Cancel」可關閉此功能表；點擊各項使用者功能則會跳轉至相對應的功能頁面。

檢視個人資料

  • 於一般使用者功能表點擊「User Profile」，可以查看個人資料，包括：帳號、姓名、Email、性別、身分、權限。
  • 若使用者身份為「Student」，則會多顯示學生基本資料，其中有：學號、科系、中隊、年班。

更新個人資料

  • 於一般使用者功能表點擊「Profile Update」，可以個人資料的更改，可更改項目有：姓名、性別、個人簡介及個人頭像。
  • 若使用者身份為「Student」，則會多顯示學生基本資料，可更改項目有：科系、中隊、年班。
  • 完成更改後，點擊「Update」即可完成更新。

Email

  • 於一般使用者功能表點擊「Email」，可進行Email的相關操作。
  • 尚未進行Email的更改，該輸入欄位會顯示「None」。
  • 已完成驗證的Email在個人資料的Email欄位會看見旁邊有。
  • 亦可進行已完成驗證的信箱進行更改，但須至電子郵件信箱進行相關驗證程序進行驗證。

更改密碼

  • 於一般使用者功能表點擊「Password Change」，可進行密碼的更改。
  • 若使用者完成密碼更改後，系統會自動登出。此時，使用者需使用新密碼重新進行登入。

問題回報

  • 於一般使用者功能表點擊「Report」，可進行在系統操作過程中所遇到的問題進行回報。
